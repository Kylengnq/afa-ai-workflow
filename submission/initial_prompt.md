# Initial Prompt

The AFA 2027 call requires that the investigation begin **on or after 2026-06-01**
with an initial prompt to an LLM or AI agent. Capture that prompt here verbatim.

## Metadata

| Field | Value |
|---|---|
| Date issued | YYYY-MM-DD (must be on or after 2026-06-01) |
| Time issued | HH:MM (local time, with timezone) |
| Model | e.g., `claude-opus-4-7`, `gpt-5`, `gemini-3-pro` |
| Interface | e.g., Claude Code CLI, Codex CLI, ChatGPT web, custom agent |
| Author who issued the prompt | name |
| Attached materials | list any files, links, datasets attached to the prompt |
| Scope or constraints given | any rules, budgets, or instructions imposed on the agent |

## Prompt text

```
Paste the exact text of the initial prompt here, including any system prompts,
attached file references, or tool-use hints. Do not paraphrase. If the prompt
referenced external files, list them and include the files (or links) under
"Attached materials" above.
```

## Why this question

One or two sentences on why this question was chosen. The call notes that "taste,
judgment, values, and intuition" remain a human contribution; this is the place
to document that contribution at project start.
